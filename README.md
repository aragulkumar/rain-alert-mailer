# rain-alert-mailer
A simple project that uses a Weather API to check rainfall and sends an email notification if rain is predicted.
Doing this project by learning from 100days of code course from udemy
This project checks the daily weather forecast using a Weather API and sends an email alert if rain is expected.

## Features
- Uses open Weather API
- Sends email notification
- Works automatically using scheduler
- Customizable location

## Setup
1. Clone the repo
2. Install dependencies
3. Add API key
4. Run the script
